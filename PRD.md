# 🪄 魔法课桌 (Magic Desk) - 产品需求文档 (PRD)

## 1. 产品愿景
为小学生提供一个充满“魔法感”的学习管理助手，通过 AI 对话和视觉识别，将枯燥的作业和课程转化为有趣的探险任务。

## 2. 核心红线 (The Golden Rules)
- **教育底线**：严禁直接给出题目答案或解题步骤。
- **引导原则**：AI 应以“导师”身份提供启发性提示，鼓励独立思考。
- **视觉风格**：深紫色调魔法风格（参考 Stitch 原型），强调趣味性和沉浸感。

## 3. 功能模块 (MVP 阶段)
- **魔法相机 (Magic Scan)**：拍照识别作业，自动拆解为任务清单。
- **魔法课程表 (Magic Schedule)**：动态展示当天的“探险节点”。
- **导师对话 (Tutor Chat)**：基于长期记忆的启发式学习辅助。
- **成就系统 (Mana/XP)**：将任务完成度转化为能量条显示。

## 4. 商业与扩展规划
- **第一阶段**：单用户本地测试，验证 Coze 后端与 Next.js 前端的联调。
- **第二阶段**：接入多用户账号体系，实现数据隔离与商业化部署。
